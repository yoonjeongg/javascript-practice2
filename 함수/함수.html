함수 정의 후 호출

전달 된 매개변수 중 최소값을 반환하는 함수 min(a, b)과 전달 된 매개변수 중 최대값을 반환하는 함수 max(a, b)를 정의하고 prompt로 숫자 2번 입력 받아 입력 받은 값의 최소값과 최대값을 함수 호출을 통해 리턴 받아 콘솔에 출력한다.

x의 n제곱을 반환해주는 함수 pow(x,n)를 정의하고 prompt로 x, n 값을 입력 받은 뒤 함수 호출하여 alert로 리턴 받은 결과 값을 출력한다. * x의 n 제곱은 x를 n번 곱해서 만들 수 있다. * 주의사항 : n은 1 이상의 자연수여야 하므로 이외의 경우엔 자연수를 입력하라는 alert를 출력한다.

화살표 함수로 변경

하단의 script의 test 함수의 인자를 question1, question2 함수와 수행 내용이 동일한 화살표 함수로 변경한다.

위에서 작성한 min, max 함수를 화살표 함수로 변경 생성 후 prompt로 숫자 2번 입력 받아 입력 받은 값의 최소값과 최대값을 함수 호출을 통해 리턴 받아 콘솔에 출력한다.
